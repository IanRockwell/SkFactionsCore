# SkFactionsCore
A completely open source Minecraft Factions Core written completely in Skript

## Features
`Sandbots` <br />
`Genbuckets` <br />

## Dependencies
`Plugins:` <br />
`  Vault` <br />
`  FactionsUUID (or a fork like Saber)` <br />
`  An economy plugin (like Essentials)` <br />
`Skript Addons:` <br />
`  Skript-YAML` <br />
`  Skript-Reflect` <br />
